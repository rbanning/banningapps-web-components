Web Components

Overview

Web Components (or Custom Elements ) allow the developer to create new HTML elements using plain JS/HTML/CSS. These custom elements allow for less repetitive code, modular development, and better maintenance.

The web components included here are provided to show a) examples of what goes into creating a custom element, and b) some quick coding by Banning Applications

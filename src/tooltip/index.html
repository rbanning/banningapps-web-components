Web Components

Tooltip

ba-tooltip

Overview

The ba-tooltip component is a basic Web Component that demonstrates how to ...

  • use attributes to control style (tip color and background)
  • use <template> to pass in HTML or basic innerText to pass in simple text
  • use public methods to expose functionality to the app

Usage

  <ba-tooltip [tip_background] [tip_color]>
    Simple text to show in tooltip
  </ba-tooltip>
or
  <ba-tooltip [tip_background] [tip_color]>
    <template>
    HTML elements to show in tooltip
    </template>
  </ba-tooltip>

Web Components

Profile Card

ba-profile-card

Overview

The ba-profile-card component is a more advanced Web Component includes attributes for setting theme, name, and image as well as <template> to include rich information about the card. The component also includes public methods that expose some of the functionality of the component (e.g. setTheme(), setFullName(), and setProfileImage()). The component also exposes the toggleDetailView() method that lets you show/hide the card's detail.

Usage

  <ba-profile-card [theme] [name] [image]>
    <template>
        <h3>Heading</h3>
        <p>
          This is where the details will go.
        </p>
    </template>
  </ba-profile-card>

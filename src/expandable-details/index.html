Web Components

Expandable Details

ba-expandable-details

Overview

The ba-expandable-details component is a basic Web Component that wraps the HTML details and summary components. It also demonstrates the use of slot elements to place content from the innerHTML of the component into the shadow DOM.

Usage

  <ba-expandable-details>
    <span slot="heading">The Heading</span>
    <span slot="content">
        <h3>HTML Here</h3>
        <p>
          You can put HTML element in the content!
        </p>
    </span>
  </ba-expandable-details>
